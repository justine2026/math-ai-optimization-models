## Technical Implementation: Linear Regression
The current model in linear_optimization.py uses *Ordinary Least Squares (OLS)* to minimize the sum of the squares of the vertical deviations between each data point and the fitted line.

### The Equation:
The predicted value ($\hat{y}$) is calculated as:
$$\hat{y} = \beta_0 + \beta_1 x$$

Where the slope ($\beta_1$) is defined by the mathematical relationship:
$$\beta_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}$$
